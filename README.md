# 7POS1 procvičování otázek

Jednoduchá statická aplikace pro procvičování otázek z Moodle XML banky.

## Použití

1. Otevři `index.html` v prohlížeči.
2. Aplikace sama načte XML soubory ze složky `questions`.
3. Test se automaticky sestaví a spustí.
4. Přepínačem vpravo nahoře můžeš měnit světlý a tmavý motiv.

## Poznámky

- Testová sada je sestavená z banky Moodle souborů v repozitáři.
- U esejí aplikace neprovádí automatické hodnocení. Zobrazí jen referenční oporu z banky.
- Pokud přidáš nebo upravíš XML soubory, stačí stránku znovu načíst.
